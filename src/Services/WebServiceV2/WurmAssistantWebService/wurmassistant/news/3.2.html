Triggers import/export!

After many eons of incredibly sophisticated development, this marvelous new feature has come to save the Wurld of Wurm Assistant!

A few triggers within any character config can now be selected and exported to a text file as JSON format. These files can then be imported into any other character config. Should importer notice a trigger already exists for this character, it will kindly ask what to do.

And for the true power users, with a bit of JSON editing and scripting, entire sets of triggers can be made without having to go through the GUI!

As an extra convenience for dealing with imported triggers, the 'Delete' keyboard key can now be used to remove all currently selected triggers at once.
